## 🚐 計畫項目

### 台灣夢音樂祭（TTL Neo-Rechao）
進駐大稻埕碼頭開唱（貨櫃市集）。搭建結合霓虹燈管、磨石子地貼與塑膠紅椅的沈浸式空間，販售限定特調。

**台灣味限定特調**
- 高粱加跳跳糖（招牌）
- 金門高粱 Sour
- 洛神花米酒頭
- 仙草凍黑糖氣泡

### 流動熱炒車（Mobile Rechao）
改裝復古電子花車，從台北大稻埕碼頭出發，巡迴六都商圈。

## 🛠️ 本地開發

```bash
# Clone 專案
git clone https://github.com/hjuming/neo-rechao.git
cd neo-rechao

# 直接用瀏覽器開啟（需要 support.js 在同一目錄）
open "Neo-Rechao 活動網站.dc.html"
```

> **注意**：`.dc.html` 檔案為 Design Component 格式，需搭配 `support.js` 執行環境才能正常顯示。建議透過本地伺服器或 Claude Design 平台開啟。
